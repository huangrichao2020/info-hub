# Serenity 卡脖子选股框架（A股适配版）

> 来源：serenity-stock-choke（@fadewalk） / 原作者：Reddit @aleabitoreddit
> 整理人：Mavis（2026-06-29）
> 用途：板块/产业链结构化分析，找"断货=地震"的瓶颈环节 → 选小盘股

---

## 0. 核心心法（一句话）

> **沿着产业链向上游追溯，找到"一旦断货，万亿产业就要地震"的关键节点——那个节点上的小盘股，就是下一个暴击机会。**

不是基本面投资，不是技术分析。**这是供应链地缘政治分析。**

### 霍尔木兹海峡类比

全球石油咽喉 → 封锁 → 万亿产业地震 → 拥有海峡股份 = 拥有定价权

**翻译到 A 股**：哪个环节是某个大赛道的"霍尔木兹"？哪个 A 股公司控制它？

---

## 1. 三段逻辑链

```
① AI大爆发 / 政策驱动 / 技术跃迁          ← 终端需求暴增
② 上游硬件/原材料需求暴增                  ← 产能跟不上
③ 瓶颈环节掌握定价权 → 小盘股弹性最大       ← 选股落脚
```

---

## 2. 六步通用推理法

### 第一步：定位板块周期（三种类型）

| 类型 | 特征 | 信号 | 典型板块 |
|---|---|---|---|
| **需求爆发期** | 终端需求暴增，中游扩产来不及 | 价格上涨，订单排几年 | [[AI 算力]] / 固态电池 |
| **技术跃迁期** | 新技术替代旧技术 | 政策打压旧的，补贴新的 | CPO / HJT |
| **供给受限期** | 关键原材料被卡 / 产能有限 | 报价暴涨，供应商份额=壁垒 | InP 衬底 / 高纯石英 |

### 第二步：溯源供应链，找卡脖子节点

```
终端产品
  ↓
[最终组装/集成]        ← 集中度风险？
  ↓
[核心零部件]          ← 技术壁垒？
  ↓
[关键材料/元器件]      ← 原料依赖？国内能否自产？
  ↓
[上游化工/矿产/稀有气体] ← 资源垄断？
```

**判断卡脖子的 6 条标准**（同时满足 3+）：
- ✅ 技术壁垒极高（专利林立、know-how 深）
- ✅ 产能建设周期长（2-5 年扩产窗口）
- ✅ 国内自给率低（依赖进口）
- ✅ 单一供应商/寡头垄断
- ✅ 不可替代性高（无备选方案）
- ✅ 地缘政治风险（出口管制 / 制裁）

### 第三步：找 A 股对应标的，建立"四维信号卡"

每个候选股必填：

| 维度 | 要查什么 | 数据来源（Mavis 适配） |
|---|---|---|
| **[[卡脖子定位]]** | 份额/技术壁垒/专利 | 公开年报 + 公告 |
| **估值水位** | P/E、P/B 在板块内分位 | 行情接口 |
| **资本信号** | 主力净流入、融资融券、龙虎榜 | eastmoney / 同花顺 |
| **机构关注度** | 近期研报覆盖 / 目标价 | 研报搜索 |

### 第四步：筛选"真瓶颈" vs "伪概念"

**6 条排除规则**（有一条就剔除）：

| # | 排除条件 | 典型案例 |
|---|---|---|
| 1 | 业务不主营（蹭热点） | 主营做电缆突然说做 CPO |
| 2 | 国内格局分散，无护城河 | 大量同质化小厂 |
| 3 | 产能扩张太容易（壁垒低） | 通用型原材料 |
| 4 | 进口替代逻辑不成立（国外也无货） | 全球都缺的氖气 |
| 5 | 估值已充分反映（过于知名） | 各环节龙头都已大涨 |
| 6 | 无机构关注（流动性陷阱） | 日均成交额 < 5000 万 |

### 第五步：多空双向确认

| 做多信号 | 做空 / 观望 |
|---|---|
| 板块主力净流入持续 | 板块主力净流出 |
| 机构开始覆盖（研报出现） | 无研报 / 无人问津 |
| 产能建设周期 > 2 年 | 产能半年可达 |
| 政策明确支持 | 政策压制 |
| 筹码集中度提升（股东减少） | 股东人数暴增 |
| 估值分位低于板块平均 | P/E > 历史 80% 分位 |

### 第六步：输出 7 段式报告

```text
## [板块名称] 卡脖子分析报告

### 1. 板块所处周期判断
### 2. 供应链卡脖子地图
### 3. 候选标的四维信号卡
### 4. 筛选结论
### 5. 多空信号对照
### 6. 今日板块资金面
### 7. 操作建议
```

## 3. A 股适配的 5 个特色维度

| 维度 | 美股原版 | A 股适配 |
|---|---|---|
| 政策权重 | 较低 | **极高**（直接催化板块） |
| 主力资金 | 对冲基金/机构 | **公募 + 私募 + 游资** |
| 散户结构 | 少 | **多**（情绪波动大） |
| 壳资源 | 无 | 小市值有壳价值溢价 |
| [[国产替代]] | 非核心 | **核心逻辑**（卡脖子清单） |

**A 股独有的 5 个信号维度**：
1. **龙虎榜**：游资席位暴露短线情绪
2. **融资融券**：杠杆资金的态度
3. **主力净流入**：>5 亿 = 机构明显建仓
4. **政府工作报告**：每年 3 月更新，方向性信号
5. **卡脖子清单**：国务院发布，指引[[国产替代]]

## 5. Serenity 风险提示（必看）

1. **波动极大**：小盘股一天可以涨跌 20-30%
2. **流动性风险**：建仓/清仓时股价剧烈波动
3. **时间成本**：逻辑验证可能需要 1-3 年
4. **信号噪音**：供应链消息真真假假，需独立验证
5. **政策异动**：A 股受政策影响极大，监管动态要密切跟踪

## 6. Mavis 适配说明

- `neodata-financial-search`（行情/研报/资金）

- 报告输出：套 trading-review 现有的"7 段式"

**触发方式**：
- "用 serenity 框架分析 [板块]"
- "[板块] 卡脖子机会"
- "[板块] 瓶颈环节有哪些"

## 7. 与 Mavis 现有方法论的关系

```text
Mavis 已有                Serenity 互补
─────────────            ─────────────
trading-review (复盘)     → 板块卡脖子分析（事前选股）
weekly-outlook (周报)     → 板块横切分析（深入颗粒度）
info-hub 主线龙头挖掘     → 全市场维度
serenity-stock-choke      → 单板块纵切深度
```

**协作方式**：
- weekly-outlook 给出 5 大方向 → 用 serenity 框架对其中 1-2 个最看好的板块纵切深挖
- info-hub 给出主线龙头 → serenity 验证是否真的在卡脖子节点上
- 复盘记录"实际走势 vs serenity 预测" → 沉淀回 knowhub
